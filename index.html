Frequently Asked Questions

1.What is LifeSteal SMP?

LifeSteal SMP is a competitive Minecraft survival server where you steal hearts by killing other players. Form alliances, raid bases, and climb the leaderboard.

2.How do I join on Java?

Open Multiplayer → Add Server, then enter the server address: lifestealsmp.com.

3.How do I join on Bedrock?

On Bedrock, use Realms → Add/join Realm → Join an existing Realm, then enter the Realm code shown on the website.

4.Java + Bedrock supported?

Yes. LifeSteal SMP supports both Java and Bedrock players.

5.Why vote for the server?

Voting helps the server grow and can provide in-game rewards depending on current events and reward settings.
